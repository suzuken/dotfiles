ncurses_panel_below

(PHP 4 >= 4.3.0, PHP 5 < 5.3.0, PECL ncurses >= 1.0.0)

ncurses_panel_below — パネルの下のパネルを返す

説明

resource ncurses_panel_below ( resource $panel )
警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

panel

パラメータ

パネルが null の場合は、スタックの最上部のパネルを返します。
