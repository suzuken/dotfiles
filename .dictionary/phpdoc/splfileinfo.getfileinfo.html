SplFileInfo::getFileInfo

(PHP 5 >= 5.1.2)

SplFileInfo::getFileInfo — ファイルの SplFileInfo オブジェクトを取得する

説明

public SplFileInfo SplFileInfo::getFileInfo ([ string $class_name ] )

このメソッドは、参照先のファイルの SplFileInfo オブジェクトを取得します。

パラメータ

class_name

使用したい SplFileInfo 派生クラスの名前。

返り値

そのファイル用に作成した SplFileInfo オブジェクトを返します。

参考

  • SplFileInfo::setInfoClass() - getFileInfo および getPathInfo で使用するクラスを設定する
