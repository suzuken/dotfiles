LimitIterator::getPosition

(PHP 5 >= 5.1.0)

LimitIterator::getPosition — 現在の位置を返す

説明

public int LimitIterator::getPosition ( void )

内部 Iterator の、ゼロから数えた現在位置を返します。

パラメータ

この関数にはパラメータはありません。

返り値

現在の位置を返します。

例

例1 LimitIterator::getPosition() の例

<?php
$fruits = array(
    'a' => 'apple',
    'b' => 'banana',
    'c' => 'cherry',
    'd' => 'damson',
    'e' => 'elderberry'
);
$array_it = new ArrayIterator($fruits);
$limit_it = new LimitIterator($array_it, 2, 3);
foreach ($limit_it as $item) {
    echo $limit_it->getPosition() . ' ' . $item . "\n";
}
?>

上の例の出力は以下となります。

2 cherry
3 damson
4 elderberry
