hw_cp

(PHP 4)

hw_cp — オブジェクトをコピーする

説明

int hw_cp ( int $connection , array $object_id_array , int $destination_id )

2番目のパラメータとして指定されたオブジェクトID を有するオブジェクト ID destination id を有するコレクションにコピーします。

パラメータ

connection

接続 ID。

object_id_array

オブジェクト ID の配列。

destination_id

コピー先のコレクション ID。

返り値

コピーされたオブジェクトの数を返します。
