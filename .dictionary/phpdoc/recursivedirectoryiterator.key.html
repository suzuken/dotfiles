RecursiveDirectoryIterator::key

(PHP 5)

RecursiveDirectoryIterator::key — 現在のディレクトリエントリのパスとファイル名を返す

説明

string RecursiveDirectoryIterator::key ( void )
警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

現在のディレクトリエントリのパスとファイル名を返します。
