SAMMessage->body

(PECL sam >= 0.1.0)

SAMMessage->body — メッセージの本文

説明

string $body;

"body" プロパティには、メッセージの本文が含まれます。 常に設定されているとは限りません。

例

例1 メッセージ本文へのテキスト文字列の設定

<?php
$msg = new SAMMessage();
$msg->body = 'This is a simple message';

?>
