HaruPage::setWordSpace

(PECL haru >= 0.0.1)

HaruPage::setWordSpace — ページの単語間隔を設定する

説明

bool HaruPage::setWordSpace ( float $word_space )

ページの単語間隔を設定します。

パラメータ

word_space

ページの単語間隔に設定する値。

返り値

成功した場合に TRUE を返します。

エラー / 例外

エラー時に HaruException をスローします。

参考

  • HaruPage::getWordSpace() - 現在の単語間隔の値を取得する
