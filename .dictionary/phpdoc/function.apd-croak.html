apd_croak

(PECL apd 0.2-0.4)

apd_croak — エラーとコールスタックをスローし、終了する

説明

void apd_croak ( string $warning [, string $delimiter ] )

perl の Carp::croak と同じように動作します。エラーとコールバックをスローし、 終了します。

パラメータ

warning

スローする警告。

delimiter

区切り文字。デフォルトは <BR />。

返り値

値を返しません。

例

例1 apd_croak() の例

<?php
apd_croak("Some Warning","<P>");
?>
