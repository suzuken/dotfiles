ReflectionClass::__clone

(PHP 5)

ReflectionClass::__clone — オブジェクトをクローンする

説明

final private void ReflectionClass::__clone ( void )

クローンします。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。
