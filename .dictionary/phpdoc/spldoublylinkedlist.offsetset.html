SplDoublyLinkedList::offsetSet

(PHP 5 >= 5.3.0)

SplDoublyLinkedList::offsetSet — 指定した $index の値を $newval に設定する

説明

void SplDoublyLinkedList::offsetSet ( mixed $index , mixed $newval )

指定した index の値を newval に設定します。

パラメータ

index

設定したいインデックス。

newval

index の新しい値。

返り値

値を返しません。
