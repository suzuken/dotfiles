swf_definebitmap

(PHP 4)

swf_definebitmap — ビットマップを定義する

説明

void swf_definebitmap ( int $objid , string $image_name )

関数 swf_definebitmap() は、 指定したイメージのビットマップを定義します。

パラメータ

objid

SWF オブジェクト ID。

image_name

GIF、JPEG、RGB あるいは FI 形式のイメージ。 このイメージは Flash JPEG または Flash カラーマップフォーマットに変換されます。

返り値

値を返しません。
