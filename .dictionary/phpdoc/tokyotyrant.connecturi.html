TokyoTyrant::connectUri

(PECL tokyo_tyrant >= 0.1.0)

TokyoTyrant::connectUri — データベースに接続する

説明

public TokyoTyrant TokyoTyrant::connectUri ( string $uri )

URI を使用してデータベースに接続します。

パラメータ

uri

データベースへの URI。たとえば tcp://localhost:1979/ のようになります。

返り値

現在のオブジェクトを返します。失敗した場合は TokyoTyrantException をスローします。

例

例1 TokyoTyrant::connectUri() の例

<?php
$tt = new TokyoTyrant();
$tt->connectUri("tcp://localhost:1978/")->put("test", "hi");
?>
