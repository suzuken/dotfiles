AMQPConnection::disconnect

(バージョン情報なし。おそらく SVN 版にしか存在しないでしょう)

AMQPConnection::disconnect — Closes the connection with the AMQP broker.

説明

public bool AMQPConnection::disconnect ( void )

This method will close an open connection with the AMQP broker.

パラメータ

void

返り値

Returns true if connection was successfully closed, false otherwise.

例

例1 AMQPConnection::disconnect() example

<?php

/* Create a new connection */
$cnn = new AMQPConnection();

// set the login details
$cnn->setLogin('mylogin');
$cnn->setPassword('mypass');

if (!$cnn->connect()) {
 throw new Exception('Could not connect');
}

// do something interesting

if (!$cnn->disconnect()) {
 throw new Exception('Could not disconnect');
}


?>
