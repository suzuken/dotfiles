swf_actiongotolabel

(PHP 4)

swf_actiongotolabel — 指定したラベルを有するフレームを表示する

説明

void swf_actiongotolabel ( string $label )

関数 swf_actiongotolabel() は、 label で指定したラベルを有するフレームを 表示した後で停止します。

パラメータ

label

フレームのラベル。

返り値

値を返しません。
