DomNode->has_attributes

(PHP 4 >= 4.1.0)

DomNode->has_attributes — ノードが属性を有しているかを調べる

説明

bool DomNode->has_attributes ( void )

この関数は、ノードが属性を有しているかを調べます。

domnode_has_child_nodes() も参照ください。
