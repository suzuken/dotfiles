MongoBinData::__construct

(PECL mongo >= 0.8.1)

MongoBinData::__construct — 新しいバイナリデータオブジェクトを作成する

説明

public MongoBinData::__construct ( string $data [, int $type = 2 ] )

新しいバイナリデータオブジェクトを作成します。

現在の BSON 仕様には、次の 5 種類のバイナリデータ型があります。 関数 (0x01), バイト配列 (0x02), UUID (0x03), MD5 (0x05), そしてユーザ定義型 (0x80) です。デフォルトの型はバイト配列 (0x02) です。 ドライバやサーバがこの型の違いによって処理を変えることはないので、 今のところこれは無意味です。0 から 255 までの任意の数を型に使うことができますが、 将来データベースが型を見てバイナリデータの処理を変えるようになる可能性を考えなければ鳴りません。

パラメータ

data

バイナリデータ。

type

データ型。

返り値

新しいバイナリデータオブジェクトを返します。
