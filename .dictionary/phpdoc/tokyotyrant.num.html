TokyoTyrant::num

(PECL tokyo_tyrant >= 0.1.0)

TokyoTyrant::num — データベース内のレコードの数を取得する

説明

public int TokyoTyrant::num ( void )

データベース内のレコードの数を返します。

パラメータ

この関数にはパラメータはありません。

返り値

データベース内のレコードの数を返します。

例

例1 TokyoTyrant::num() の例

<?php
/* データベースのデフォルトポートに接続します */
$tt = new TokyoTyrant("localhost");

/* レコード数を表示します */
echo $tt->num();
?>

上の例の出力は、 たとえば以下のようになります。

1234
