TokyoTyrant::stat

(PECL tokyo_tyrant >= 0.1.0)

TokyoTyrant::stat — 統計情報を取得する

説明

public array TokyoTyrant::stat ( void )

リモートデータベースの統計情報を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

統計情報をあらわす key-value ペアの配列を返します。

例

例1 TokyoTyrant::stat() の例

<?php
$tt = new TokyoTyrant("localhost");

var_dump($tt->stat());
?>
