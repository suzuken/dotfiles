unregister_tick_function

(PHP 4 >= 4.0.3, PHP 5)

unregister_tick_function — 各 tick の実行用の関数の登録を解除する

説明

void unregister_tick_function ( string $function_name )

function_name という名前の関数の登録を解除します。 これにより、tick がコールされた 場合でもこの関数は実行されなくなります。

パラメータ

function_name

関数名を表す文字列。

返り値

値を返しません。
