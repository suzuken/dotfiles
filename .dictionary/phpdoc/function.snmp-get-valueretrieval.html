snmp_get_valueretrieval

(PHP 4 >= 4.3.3, PHP 5)

snmp_get_valueretrieval — SNMP の値が返される方法を返す

説明

int snmp_get_valueretrieval ( void )

返り値

定数 SNMP_VALUE_LIBRARY, SNMP_VALUE_PLAIN, SNMP_VALUE_OBJECT のいずれかを返します。

例

例1 snmp_get_valueretrieval の使用法

<?php
 $ret = snmpget('localhost', 'public', 'IF-MIB::ifName.1');
 if (snmp_get_valueretrieval() == SNMP_VALUE_OBJECT) {
   echo $ret->value;
 } else {
   echo $ret;
 }
