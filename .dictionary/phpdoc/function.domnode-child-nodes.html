DomNode->child_nodes

(PHP 4 >= 4.1.0)

DomNode->child_nodes — 子ノードを返す

説明

array DomNode->child_nodes ( void )

全ての子ノードを返します。

domnode_next_sibling(), domnode_previous_sibling() も参照ください。
