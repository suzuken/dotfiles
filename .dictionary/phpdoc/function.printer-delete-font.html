printer_delete_font

(PECL printer SVN)

printer_delete_font — フォントを削除する

説明

void printer_delete_font ( resource $font_handle )

この関数は選択したフォントを削除します。使用例は printer_select_font() を参照ください。

パラメータ

font_handle

font_handle は、 有効なフォントのハンドルである必要があります。

返り値

値を返しません。
