ArrayIterator::key

(PHP 5 >= 5.0.0)

ArrayIterator::key — 現在の配列キーを返す

説明

mixed ArrayIterator::key ( void )

この関数は、現在の配列キーを返します

パラメータ

この関数にはパラメータはありません。

返り値

この関数は、現在の配列キーを返します

例

例1 ArrayIterator::key() の例

<?php
$array = array('key' => 'value');

$arrayobject = new ArrayObject($array);
$iterator = $arrayobject->getIterator();

echo $iterator->key(); //key
?>
