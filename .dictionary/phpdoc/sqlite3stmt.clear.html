SQLite3Stmt::clear

(PHP 5 >= 5.3.0)

SQLite3Stmt::clear — 現在バインドされているすべてのパラメータをクリアする

説明

public bool SQLite3Stmt::clear ( void )

現在バインドされているすべてのパラメータをクリアします。

パラメータ

この関数にはパラメータはありません。

返り値

バインドされているパラメータのクリアに成功した場合に TRUE、 失敗した場合に FALSE を返します。
