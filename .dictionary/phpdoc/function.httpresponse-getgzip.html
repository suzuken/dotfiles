HttpResponse::getGzip

(PECL pecl_http >= 0.10.0)

HttpResponse::getGzip — gzip を取得する

説明

static bool HttpResponse::getGzip ( void )

現在の gzip 圧縮設定を取得します。

返り値

GZip 圧縮が有効な場合に TRUE、そうでない場合に FALSE を返します。

参考

  • HttpResponse::setGzip() - gzip を設定する
