GearmanWorker::error

(PECL gearman >= 0.5.0)

GearmanWorker::error — Get the last error encountered

説明

public string GearmanWorker::error ( void )

Returns an error string for the last error encountered.

パラメータ

この関数にはパラメータはありません。

返り値

An error string.
