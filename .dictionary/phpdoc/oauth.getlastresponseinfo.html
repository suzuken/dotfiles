OAuth::getLastResponseInfo

(PECL OAuth >= 0.99.1)

OAuth::getLastResponseInfo — 直近のレスポンスの HTTP 情報を取得する

説明

public array OAuth::getLastResponseInfo ( void )

直近のレスポンスに関する HTTP 情報を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

直近のリクエストに対するレスポンスの情報を含む配列を返します。 curl_getinfo() の定数が使われます。

参考

  • OAuth::fetch() - OAuth で保護されたリソースを取得する
  • OAuth::getLastResponse() - 直近のレスポンスを取得する
