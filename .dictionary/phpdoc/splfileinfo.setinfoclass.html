SplFileInfo::setInfoClass

(PHP 5 >= 5.1.2)

SplFileInfo::setInfoClass — getFileInfo および getPathInfo で使用するクラスを設定する

説明

public void SplFileInfo::setInfoClass ([ string $class_name ] )

このメソッドで、getFileInfo および getPathInfo がコールされた際に使用するカスタムクラスを設定します。 このメソッドに渡すクラスは、SplFileInfo の派生クラスでなければなりません。

パラメータ

class_name

使用するクラス名。

返り値

値を返しません。

例

例1 SplFileInfo::setFileClass() の例

<?php
// SplFileInfo を継承したクラスを定義します
class MyFoo extends SplFileInfo {}

$info = new SplFileInfo('foo');
// 使用するクラス名を設定します
$info->setInfoClass('MyFoo');
var_dump($info->getFileInfo());
?>

上の例の出力は、 たとえば以下のようになります。

object(MyFoo)#2 (0) { }
