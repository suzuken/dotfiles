mcrypt_enc_get_block_size

(PHP 4 >= 4.0.2, PHP 5)

mcrypt_enc_get_block_size — オープンされたアルゴリズムのブロックサイズを返す

説明

int mcrypt_enc_get_block_size ( resource $td )

オープンされたアルゴリズムのブロックサイズを取得します。

パラメータ

td

暗号化記述子。

返り値

指定したアルゴリズムのブロック長をバイト単位で返します。
