FilterIterator::current

(PHP 5 >= 5.1.0)

FilterIterator::current — 現在の要素の値を取得する

説明

mixed FilterIterator::current ( void )
警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

現在の要素の値を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

現在の要素の値を返します。

参考

  • FilterIterator::key() - 現在のキーを取得する
  • FilterIterator::next() - イテレータを前に移動する
