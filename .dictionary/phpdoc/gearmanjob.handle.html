GearmanJob::handle

(PECL gearman >= 0.5.0)

GearmanJob::handle — Get the job handle

説明

public string GearmanJob::handle ( void )

Returns the opaque job handle assigned by the job server.

パラメータ

この関数にはパラメータはありません。

返り値

An opaque job handle.
