apd_dump_regular_resources

(PECL apd 0.2-0.4)

apd_dump_regular_resources — 現在のすべての一般リソースを配列で返す

説明

array apd_dump_regular_resources ( void )

現在のすべての一般リソースを配列で返します。

返り値

現在の一般リソースを含む配列を返します。

例

例1 apd_dump_regular_resources() の例

<?php
print_r(apd_dump_regular_resources());
?>
