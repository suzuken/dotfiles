printer_abort

(PECL printer SVN)

printer_abort — プリンタのスプールファイルを削除する

説明

void printer_abort ( resource $printer_handle )

この関数は、プリンタのスプールファイルを削除します。

パラメータ

printer_handle

printer_handle には、プリンタへの有効なハンドルを指定する必要があります。

返り値

値を返しません。

例

例1 printer_abort() の例

<?php
$handle = printer_open();
printer_abort($handle);
printer_close($handle);
?>
