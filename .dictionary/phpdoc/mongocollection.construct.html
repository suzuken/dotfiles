MongoCollection::__construct

(PECL mongo >=0.9.0)

MongoCollection::__construct — 新しいコレクションを作成する

説明

public MongoCollection::__construct ( MongoDB $db , string $name )

パラメータ

MongoDB db

親データベース。

name

コレクション名。

返り値

新しいコレクションオブジェクトを返します。

エラー / 例外

コレクション名が不正な場合に、デフォルトの例外をスローします。
