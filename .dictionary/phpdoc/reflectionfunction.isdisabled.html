ReflectionFunction::isDisabled

(PHP 5)

ReflectionFunction::isDisabled — 関数が無効になっているかどうかを調べる

説明

public bool ReflectionFunction::isDisabled ( void )

disable_functions ディレクティブで関数が無効になっているかどうかを調べます。

パラメータ

この関数にはパラメータはありません。

返り値

無効になっている場合に TRUE、それ以外の場合に FALSE を返します。

参考

  • ReflectionFunctionAbstract::isUserDefined() - ユーザ定義関数であるかどうかを調べる
  • disable_functions ディレクティブ
