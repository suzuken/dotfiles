hwapi_hgcsp

(PHP 4, PHP 5 < 5.2.0, PECL hwapi SVN)

hwapi_hgcsp — hw_api クラスのオブジェクトを返す

説明

HW_API hwapi_hgcsp ( string $hostname [, int $port ] )

ホスト hostname 上の Hyperwave サーバへの接続をオープンします。 プロトコルは HGCSP を使用します。

パラメータ

hostname

ホスト名。

port

ポート番号を渡さなかった場合は、418 が用いられます。

返り値

HW_API のインスタンスを返します。
