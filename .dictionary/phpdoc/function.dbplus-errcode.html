dbplus_errcode

(PHP 4 <= 4.1.0, PECL dbplus >= 0.9)

dbplus_errcode — 指定したエラーコードまたは直近のエラーに関するエラー文字列を取得する

説明

string dbplus_errcode ([ int $errno ] )

指定したエラーコードに対応するエラー文字列を返します。

パラメータ

errno

エラーコード。指定しなかった場合は、直近の db++ 操作の結果コードを使用します。

返り値

エラーメッセージを返します。
