domxml_version

(PHP 4 >= 4.1.0)

domxml_version — XML ライブラリのバージョンを取得する

説明

string domxml_version ( void )

この関数は、現在使用されている XML ライブラリのバージョンを取得します。

返り値

XML ライブラリのバージョンを文字列として返します。

例

例1 domxml_version() の例

<?php

echo domxml_version();

?>

上の例の出力は、 たとえば以下のようになります。

20607
