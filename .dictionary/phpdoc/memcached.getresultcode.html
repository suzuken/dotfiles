Memcached::getResultCode

(PECL memcached >= 0.1.0)

Memcached::getResultCode — 直近の操作の結果コードを返す

説明

public int Memcached::getResultCode ( void )

Memcached::getResultCode() は Memcached::RES_* 定数のいずれかを返します。 これは、直近に実行された Memcached メソッドの結果を表します。

パラメータ

この関数にはパラメータはありません。

返り値

直近の Memcached 操作の結果コードを返します。

例

例1 Memcached::getResultCode() の例

<?php
$m = new Memcached();
$m->addServer('localhost', 11211);

$m->add('foo', 'bar');
if ($m->getResultCode() == Memcached::RES_NOTSTORED) {
    /* ... */
}
?>
