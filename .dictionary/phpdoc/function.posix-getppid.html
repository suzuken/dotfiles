posix_getppid

(PHP 4, PHP 5)

posix_getppid — 親プロセスの ID を返す

説明

int posix_getppid ( void )

現在のプロセスの親プロセスの ID を返します。

返り値

ID を表す整数値を返します。

例

例1 posix_getppid() の使用例

<?php
echo posix_getppid(); //8259
?>
