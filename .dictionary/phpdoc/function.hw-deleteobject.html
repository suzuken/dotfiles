hw_Deleteobject

(PHP 4)

hw_Deleteobject — オブジェクトを削除する

説明

bool hw_deleteobject ( int $connection , int $object_to_delete )

2 番目のパラメータで指定されたオブジェクト ID を有するオブジェクトを 削除します。この関数は、オブジェクトの全てのインスタンスを削除します。

パラメータ

connection

接続 ID。

object_to_delete

オブジェクト ID。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
