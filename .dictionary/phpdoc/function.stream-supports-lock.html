stream_supports_lock

(PHP 5 >= 5.3.0)

stream_supports_lock — ストリームがロックをサポートしているかどうかを調べる

説明

bool stream_supports_lock ( resource $stream )

flock() によるロックをそのストリームがサポートしているかどうかを調べます。

パラメータ

stream

調べたいストリーム。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。

参考

  • flock() - 汎用のファイルロックを行う
