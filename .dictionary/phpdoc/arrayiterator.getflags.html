ArrayIterator::getFlags

(PHP 5 >= 5.1.0)

ArrayIterator::getFlags — フラグを取得する

説明

public void ArrayIterator::getFlags ( void )

現在のフラグを取得します。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

現在のフラグを返します。
