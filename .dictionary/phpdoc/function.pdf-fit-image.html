PDF_fit_image

(PECL pdflib >= 2.0.0)

PDF_fit_image — 画像やテンプレートを配置する

説明

bool PDF_fit_image ( resource $pdfdoc , int $image , float $x , float $y , string $optlist )

画像やテンプレートを、さまざまなオプションに基づいてページ上に配置します。 成功した場合に TRUE を、失敗した場合に FALSE を返します。
