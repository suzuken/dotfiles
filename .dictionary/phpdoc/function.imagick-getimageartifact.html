Imagick::getImageArtifact

(バージョン情報なし。おそらく SVN 版にしか存在しないでしょう)

Imagick::getImageArtifact — 画像のアーチファクトを取得する

説明

string Imagick::getImageArtifact ( string $artifact )

画像に関連づけられたアーチファクトを取得します。プロパティとアーチファクトの違いは、 プロパティは public であるのに対してアーチファクトは private であるという点です。 このメソッドは、ImageMagick バージョン 6.5.7 以降で Imagick をコンパイルした場合に使用可能です。

パラメータ

artifact

アーチファクトの名前。

返り値

成功した場合にアーチファクトの値を返します。 エラー時に ImagickException をスローします。

参考

  • Imagick::setImageArtifact() - 画像のアーチファクトを設定する
  • Imagick::deleteImageArtifact() - 画像のアーチファクトを削除する
