printer_delete_brush

(PECL printer SVN)

printer_delete_brush — ブラシを削除する

説明

void printer_delete_brush ( resource $brush_handle )

この関数は、選択したブラシを削除します。使用例は printer_select_brush() を参照ください。

パラメータ

brush_handle

brush_handle は、 有効なブラシのハンドルである必要があります。

返り値

値を返しません。
