ImagickPixel::clear

(PECL imagick 2.0.0)

ImagickPixel::clear — このオブジェクトに関連付けられたリソースを消去する

説明

bool ImagickPixel::clear ( void )
警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

ImagickPixel オブジェクトをクリアし、初期状態に戻します。 このオブジェクトに関連付けられている色も初期化します。

返り値

成功した場合に TRUE を返します。
