mcrypt_module_close

(PHP 4 >= 4.0.2, PHP 5)

mcrypt_module_close — mcrypt モジュールを閉じる

説明

bool mcrypt_module_close ( resource $td )

指定した暗号化ハンドルを閉じます。

パラメータ

td

暗号化記述子。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
