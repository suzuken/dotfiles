HaruFont::getDescent

(PECL haru >= 0.0.1)

HaruFont::getDescent — フォントの垂直 descent を取得する

説明

int HaruFont::getDescent ( void )

フォントの垂直 descent (ベースラインの下側の高さ) を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

フォントの垂直 descent (ベースラインの下側の高さ) を返します。

エラー / 例外

エラー時に HaruException をスローします。
