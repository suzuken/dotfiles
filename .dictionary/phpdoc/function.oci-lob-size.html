OCI-Lob->size

(PHP 5, PECL OCI8 >= 1.1.0)

OCI-Lob->size — ラージオブジェクトのサイズを返す

説明

int OCI-Lob::size ( void )

ラージオブジェクトのサイズを取得します。

返り値

ラージオブジェクトのサイズを返します。 失敗した場合に FALSE を返します。 空のオブジェクトは長さ 0 となります。
