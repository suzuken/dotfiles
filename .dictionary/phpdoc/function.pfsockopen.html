pfsockopen

(PHP 4, PHP 5)

pfsockopen — 持続的な Internet または Unix ドメインソケット接続をオープンする

説明

resource pfsockopen ( string $hostname [, int $port = -1 [, int &$errno [, string &$errstr [, float $timeout = ini_get("default_socket_timeout") ]]]] )

この関数は、fsockopen() と全く同様に動作します。 ただし、この関数による接続はリクエストが終了した後でも閉じられないという違いがあります。 この関数は、fsockopen() の持続的接続版です。

パラメータ

パラメータについての情報は fsockopen() のドキュメントを参照ください。
