Collator::getErrorCode

collator_get_error_code

(PHP 5 >= 5.3.0, PECL intl >= 1.0.0)

Collator::getErrorCode -- collator_get_error_code — collator の直近のエラーコードを取得する

説明

オブジェクト指向型

int Collator::getErrorCode ( void )

手続き型

int collator_get_error_code ( Collator $coll )

パラメータ

coll

Collator オブジェクト。

返り値

直近の Collator API 関数コールが返したエラーコードを返します。

例

例1 collator_get_error_code() の例

<?php
$coll = collator_create( 'en_US' );
if( collator_get_attribute( $coll, Collator::FRENCH_COLLATION ) === false )
        handle_error( collator_get_error_code() );
?>
