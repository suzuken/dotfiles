ImagickDraw::getFillRule

(PECL imagick 2.0.0)

ImagickDraw::getFillRule — 塗りつぶしルールを返す

説明

int ImagickDraw::getFillRule ( void )
警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

多角形の描画時に使用する塗りつぶしルールを返します。

返り値

FILLRULE_ 定数を返します。
