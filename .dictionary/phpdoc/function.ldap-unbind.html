ldap_unbind

(PHP 4, PHP 5)

ldap_unbind — LDAP ディレクトリへのバインドを解除する

説明

bool ldap_unbind ( resource $link_identifier )

LDAP ディレクトリへのバインドを解除します。

パラメータ

link_identifier

ldap_connect() が返す LDAP リンク ID。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
