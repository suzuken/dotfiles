PDF_info_textline

(PECL pdflib >= 2.1.0)

PDF_info_textline — テキストの行のフォーマットを行い、メトリクスを問い合わせる

説明

float PDF_info_textline ( resource $pdfdoc , string $text , string $keyword , string $optlist )

テキストの行のフォーマットを行い、その結果のメトリクスを問い合わせます。
