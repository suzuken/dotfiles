session_module_name

(PHP 4, PHP 5)

session_module_name — 現在のセッションモジュールを取得または設定する

説明

string session_module_name ([ string $module ] )

session_module_name()は、 現在のセッションモジュールの名前を返します。

パラメータ

module

module が指定された場合、 そのモジュールを代わりに使用します。

返り値

現在のセッションモジュールの名前を返します。
