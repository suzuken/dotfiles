array_product

(PHP 5 >= 5.1.0)

array_product — 配列の値の積を計算する

説明

number array_product ( array $array )

array_product() は、配列の値の積を計算します。

パラメータ

array

配列。

返り値

積を、integer あるいは float で返します。

例

例1 array_product() の例

<?php

$a = array(2, 4, 6, 8);
echo "product(a) = " . array_product($a) . "\n";

?>

上の例の出力は以下となります。

product(a) = 384
