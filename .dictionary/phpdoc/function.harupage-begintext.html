HaruPage::beginText

(PECL haru >= 0.0.1)

HaruPage::beginText — テキストオブジェクトを開始し、テキストの現在位置を (0,0) に設定する

説明

bool HaruPage::beginText ( void )

新しいテキストオブジェクトを開始し、テキストの現在位置を (0,0) に設定します。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に TRUE を返します。

エラー / 例外

エラー時に HaruException をスローします。
