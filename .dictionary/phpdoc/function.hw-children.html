hw_Children

(PHP 4)

hw_Children — 子のオブジェクト ID

説明

array hw_children ( int $connection , int $objectID )

コレクションの子の ID を返します。

パラメータ

connection

接続 ID。

objectID

オブジェクト ID。

返り値

オブジェクト ID の配列を返します。各 ID は、ID objectID を有するコレクションの子に 属しています。 配列は、ドキュメントおよびコレクションの全ての子を有しています。
