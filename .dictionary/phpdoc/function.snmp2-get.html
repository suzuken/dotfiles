(バージョン情報なし。おそらく SVN 版にしか存在しないでしょう)

snmp2_get — SNMP オブジェクトを取得する

説明

string snmp2_get ( string $host , string $community , string $object_id [, string $timeout = 1000000 [, string $retries = 5 ]] )

snmp2_get() 関数は、 object_id で指定した SNMP オブジェクトの値を取得します。

パラメータ

host

SNMP エージェント。

community

リードコミュニティ。

object_id

SNMP オブジェクト。

timeout

最初のタイムアウトまでのマイクロ秒数。

retries

タイムアウト発生時の再試行回数。

返り値

成功した場合に SNMP オブジェクトの値、エラー時に FALSE を返します。

例

例1 snmp2_get() の使用法

<?php
$syscontact = snmp2_get("127.0.0.1", "public", "system.SysContact.0");
?>
