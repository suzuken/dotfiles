SplSubject::notify

(PHP 5 >= 5.1.0)

SplSubject::notify — observer に通知する

説明

abstract public void SplSubject::notify ( void )

アタッチされたすべての observer に通知します。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。
