SplFileInfo::getPath

(PHP 5 >= 5.1.2)

SplFileInfo::getPath — ファイル名を除いたパスを取得する

説明

public string SplFileInfo::getPath ( void )

ファイルへのパスを返します。ファイル名や最後のスラッシュは取り除きます。

パラメータ

この関数にはパラメータはありません。

返り値

ファイルへのパスを返します。

例

例1 SplFileInfo::getPath() の例

<?php
$info = new SplFileInfo('/usr/bin/php');
var_dump($info->getPath());


$info = new SplFileInfo('/usr/');
var_dump($info->getPath());?>

上の例の出力は、 たとえば以下のようになります。

string(8) "/usr/bin"
string(4) "/usr"
