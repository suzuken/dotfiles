printer_delete_pen

(PECL printer SVN)

printer_delete_pen — ペンを削除する

説明

void printer_delete_pen ( resource $pen_handle )

この関数は、選択したペンを削除します。使用例は printer_select_pen() を参照ください。

パラメータ

pen_handle

pen_handle は、 有効なペンのハンドルである必要があります。

返り値

値を返しません。
