readline_add_history

(PHP 4, PHP 5)

readline_add_history — ヒストリに 1 行追加する

説明

bool readline_add_history ( string $line )

この関数は、コマンドラインヒストリに 1 行追加します。

パラメータ

line

ヒストリに追加する行。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
