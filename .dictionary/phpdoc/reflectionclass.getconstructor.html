ReflectionClass::getConstructor

(PHP 5)

ReflectionClass::getConstructor — コンストラクタを取得する

説明

public object ReflectionClass::getConstructor ( void )

クラスのコンストラクタを取得します。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

ReflectionMethod オブジェクトを返します。
