HttpRequest::setMethod

(PECL pecl_http >= 0.10.0)

HttpRequest::setMethod — メソッドを設定する

説明

public bool HttpRequest::setMethod ( int $request_method )

リクエストメソッドを設定します。

パラメータ

request_method

使用するリクエストメソッド。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
