PDF_set_value

(PHP 4 >= 4.0.1, PECL pdflib >= 1.0.0)

PDF_set_value — 数値パラメータを設定する

説明

bool PDF_set_value ( resource $p , string $key , float $value )

PDFlib パラメータの値を数値型で設定します。成功した場合に TRUE を、失敗した場合に FALSE を返します。
