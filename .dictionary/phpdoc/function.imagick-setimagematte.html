Imagick::setImageMatte

(PECL imagick 2.0.0)

Imagick::setImageMatte — 画像のマットチャネルを設定する

説明

bool Imagick::setImageMatte ( bool $matte )

画像のマットチャネルを設定します。 このメソッドは、ImageMagick バージョン 6.2.9 以降で Imagick をコンパイルした場合に使用可能です。

パラメータ

matte

True にするとマットチャネルを有効にし、false の場合は無効にします。

返り値

成功した場合に TRUE を返します。
