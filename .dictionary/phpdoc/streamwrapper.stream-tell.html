streamWrapper::stream_tell

(PHP 4 >= 4.3.2, PHP 5)

streamWrapper::stream_tell — ストリームの現在位置を取得する

説明

public int streamWrapper::stream_tell ( void )

このメソッドは、ftell() に対応してコールされます。

パラメータ

この関数にはパラメータはありません。

返り値

ストリームの現在位置を返します。

参考

  • ftell() - ファイルの読み書き用ポインタの現在位置を返す
