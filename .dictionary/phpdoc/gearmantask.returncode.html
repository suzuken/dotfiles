GearmanTask::returnCode

(PECL gearman >= 0.5.0)

GearmanTask::returnCode — Get the last return code

説明

public int GearmanTask::returnCode ( void )

Returns the last Gearman return code for this task.

パラメータ

この関数にはパラメータはありません。

返り値

A valid Gearman return code.
