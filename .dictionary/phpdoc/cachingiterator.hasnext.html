CachingIterator::hasNext

(PHP 5)

CachingIterator::hasNext — 内部イテレータが有効な次の要素を持つかどうかをチェックする

説明

public void CachingIterator::hasNext ( void )
警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
