OAuth::disableRedirects

(PECL OAuth >= 0.99.9)

OAuth::disableRedirects — リダイレクトを無効にする

説明

public bool OAuth::disableRedirects ( void )

リダイレクト先を自動的にたどる機能を無効にし、リクエストを手動でリダイレクトできるようにします。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

TRUE
