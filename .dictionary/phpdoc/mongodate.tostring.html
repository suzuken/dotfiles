MongoDate::__toString

(PECL mongo >= 0.8.1)

MongoDate::__toString — この日付の文字列表現を返す

説明

publicstring MongoDate::__toString ( void )

この日付の文字列表現を返します。microtime() が返す形式と同じようになります。

パラメータ

この関数にはパラメータはありません。

返り値

この日付を返します。
