ps_setgray

(PECL ps >= 1.1.0)

ps_setgray — グレー値を設定する

説明

bool ps_setgray ( resource $psdoc , float $gray )

以降の描画処理で使われるグレー値を設定します。

パラメータ

psdoc

ps_new() が返す、postscript ファイルのリソース ID。

gray

値は 0（白）から 1（黒）までの間でなければなりません。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
