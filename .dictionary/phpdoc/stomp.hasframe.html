Stomp::hasFrame

stomp_has_frame

(PECL stomp >= 0.1.0)

Stomp::hasFrame -- stomp_has_frame — Indicates whether or not there is a frame ready to read

説明

Object oriented style (method):

public bool Stomp::hasFrame ( void )

Procedural style:

bool stomp_has_frame ( resource $link )

Indicates whether or not there is a frame ready to read.

パラメータ

link

手続き型のみ: stomp_connect() が返す stomp リンク ID。

返り値

Returns TRUE if a frame is ready to read, or FALSE otherwise.
