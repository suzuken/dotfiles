Judy::getType

(PECL judy >= 0.1.1)

Judy::getType — Return the type of the current Judy array

説明

public int Judy::getType ( void )

Return an integer corresponding to the Judy type of the current object.

パラメータ

この関数にはパラメータはありません。

返り値

Return an integer corresponding to a Judy type.
