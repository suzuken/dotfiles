MongoCursor::__construct

(PECL mongo >=0.9.0)

MongoCursor::__construct — 新しいカーソルを作成する

説明

MongoCursor::__construct ( Mongo $connection , string $ns [, array $query = array() [, array $fields = array() ]] )

パラメータ

connection

データベース接続。

ns

データベースおよびコレクションのフルネーム。

query

データベースクエリ。

fields

返すフィールド。

返り値

新しいカーソルを返します。
