newt_refresh

(PECL newt >= 0.1)

newt_refresh — 画面の変更された部分を更新する

説明

void newt_refresh ( void )

パフォーマンスを向上させるため、newt は必要時にしか画面を更新しません。 それはプログラムが端末への書き込みを行った場合とは限りません。 変更した部分を即時に更新させるようにするために、この関数をコールします。

返り値

値を返しません。
