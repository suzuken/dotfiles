GearmanWorker::options

(PECL gearman >= 0.6.0)

GearmanWorker::options — Get worker options

説明

public int GearmanWorker::options ( void )

Gets the options previously set for the worker.

パラメータ

この関数にはパラメータはありません。

返り値

The options currently set for the worker.
