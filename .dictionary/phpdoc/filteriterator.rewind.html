FilterIterator::rewind

(PHP 5 >= 5.1.0)

FilterIterator::rewind — イテレータを巻き戻す

説明

void FilterIterator::rewind ( void )
警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

イテレータを巻き戻します。

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。

  • FilterIterator::key() - 現在のキーを取得する
  • FilterIterator::next() - イテレータを前に移動する
