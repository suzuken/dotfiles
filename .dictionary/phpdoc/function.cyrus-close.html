cyrus_close

(PHP 4 >= 4.1.0, PECL cyrus 1.0)

cyrus_close — Cyrus IMAP サーバへの接続を閉じる

説明

bool cyrus_close ( resource $connection )

Cyrus IMAP サーバへの接続を閉じます。

パラメータ

connection

接続ハンドル。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
