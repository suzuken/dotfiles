ps_end_pattern

(PECL ps >= 1.2.0)

ps_end_pattern — パターンを終了する

説明

bool ps_end_pattern ( resource $psdoc )

ps_begin_pattern() によって開始したパターンを終了します。 パターンが終了すると、領域の塗りつぶし時にそれを使用できるようになります。

パラメータ

psdoc

ps_new() が返す、postscript ファイルのリソース ID。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
