libxml_clear_errors

(PHP 5 >= 5.1.0)

libxml_clear_errors — libxmlエラーハンドラをクリアする

説明

void libxml_clear_errors ( void )

libxml_clear_errors()は、libxmlエラーバッファをクリアします。

返り値

値を返しません。
