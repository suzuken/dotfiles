Imagick::decipherImage

(バージョン情報なし。おそらく SVN 版にしか存在しないでしょう)

Imagick::decipherImage — 画像を復号する

説明

bool Imagick::decipherImage ( string $passphrase )

暗号化された画像を復号します。画像の暗号化は Imagick::encipherImage() で行わなければ鳴りません。 このメソッドは、ImageMagick バージョン 6.3.9 以降で Imagick をコンパイルした場合に使用可能です。

パラメータ

passphrase

パスフレーズ。

返り値

成功した場合に TRUE を返します。
