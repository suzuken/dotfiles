EmptyIterator::current

(PHP 5 >= 5.1.0)

EmptyIterator::current — current() メソッド

説明

public void EmptyIterator::current ( void )

このメソッドをコールしてはいけません。アクセスされると例外をスローします。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

エラー / 例外

コールされると Exception をスローします。

返り値

値を返しません。
