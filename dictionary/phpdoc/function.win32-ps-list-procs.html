win32_ps_list_procs

(PECL win32ps >= 1.0.1)

win32_ps_list_procs — 稼動中のプロセスの一覧を取得する

説明

array win32_ps_list_procs ( void )

稼動しているすべてのプロセスについての統計データを取得します。

返り値

失敗した場合、FALSE を返します。 成功した場合、実行中のすべてのプロセスについての統計情報を 含む配列を返します。統計情報の形式は win32_ps_stat_proc() が返すものと同じです。
