SplFixedArray::toArray

(PHP 5 >= 5.3.0)

SplFixedArray::toArray — 固定長配列から PHP の配列を返す

説明

public array SplFixedArray::toArray ( void )

固定長配列から PHP の配列を返します。

パラメータ

この関数にはパラメータはありません。

返り値

固定長配列と同じ内容の PHP の配列を返します。

例

例1 SplFixedArray::toArray() の例

<?php
$fa = new SplFixedArray(3);
$fa[0] = 0;
$fa[2] = 2;
var_dump($fa->toArray());
?>

上の例の出力は以下となります。

array(3) {
  [0]=>
  int(0)
  [1]=>
  NULL
  [2]=>
  int(2)
}
