InfiniteIterator::next

(PHP 5 >= 5.1.0)

InfiniteIterator::next — 内部イテレータを次に移動するか、巻き戻す

説明

public void InfiniteIterator::next ( void )

次の要素があるときは、イテレータを次の要素に移動します。 要素がないときは、内部イテレータを開始位置に巻き戻します。

注意:

内部イテレータが空の場合は InfiniteIterator も停止します。

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。
