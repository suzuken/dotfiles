IteratorIterator::rewind

(PHP 5 >= 5.1.0)

IteratorIterator::rewind — 最初の要素に巻き戻す

説明

public void IteratorIterator::rewind ( void )

最初の要素に巻き戻します。

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。

参考

  • IteratorIterator::next() - 次の要素に移動する
