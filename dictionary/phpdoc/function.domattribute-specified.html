DomAttribute->specified

(PHP 4 >= 4.1.0)

DomAttribute->specified — 属性が指定されているかどうか調べる

説明

bool DomAttribute::specified ( void )

この関数は属性値を設定します。

パラメータ

content

新しい値

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。

PHP 5 への移行

DOMAttr の value プロパティを設定してください。 .
