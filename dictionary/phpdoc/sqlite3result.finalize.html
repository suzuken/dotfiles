SQLite3Result::finalize

(PHP 5 >= 5.3.0)

SQLite3Result::finalize — 結果セットを閉じる

説明

public bool SQLite3Result::finalize ( void )

結果セットを閉じます。

パラメータ

この関数にはパラメータはありません。

返り値

TRUE を返します。
