ArrayIterator::uksort

(PHP 5 >= 5.2.0)

ArrayIterator::uksort — ユーザ定義のソート

説明

public void ArrayIterator::uksort ( string $cmp_function )

ユーザ定義の関数を使って、キーでエントリをソートします。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

cmp_function

ソートに使う比較関数。

返り値

値を返しません。

参考

  • ArrayIterator::uasort() - ユーザ定義のソート
  • uksort() - ユーザ定義の比較関数を用いて、キーで配列をソートする
