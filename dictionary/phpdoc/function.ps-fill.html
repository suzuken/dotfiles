ps_fill

(PECL ps >= 1.1.0)

ps_fill — 現在のパスを塗りつぶす

説明

bool ps_fill ( resource $psdoc )

ps_lineto() のような描画関数によって事前に作成されたパスを塗りつぶします。

パラメータ

psdoc

ps_new() が返す、postscript ファイルのリソース ID。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。

参考

  • ps_fill_stroke() - 現在のパスを塗りつぶし、線を引く
  • ps_stroke() - 現在のパスを描画する
