SplFileInfo::isExecutable

(PHP 5 >= 5.1.2)

SplFileInfo::isExecutable — ファイルが実行可能かどうかを調べる

説明

public bool SplFileInfo::isExecutable ( void )

ファイルが実行可能かどうかを調べます。

パラメータ

この関数にはパラメータはありません。

返り値

実行可能である場合に TRUE、そうでない場合に FALSE を返します。

例

例1 SplFileInfo::isExecutable() の例

<?php
$info = new SplFileInfo('/usr/bin/php');
var_dump($info->isExecutable()); 

$info = new SplFileInfo('/usr/bin');
var_dump($info->isExecutable());

$info = new SplFileInfo('foo');
var_dump($info->isExecutable());
?>

上の例の出力は、 たとえば以下のようになります。

bool(true)
bool(true)
bool(false)
