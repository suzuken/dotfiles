DOMNode::hasAttributes

(PHP 5)

DOMNode::hasAttributes — ノードが属性を保持しているかどうかを調べる

説明

bool DOMNode::hasAttributes ( void )

このメソッドは、ノードが属性を保持しているかどうかを調べます。 調べるノードは XML_ELEMENT_NODE である必要があります。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
