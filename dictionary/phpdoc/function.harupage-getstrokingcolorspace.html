HaruPage::getStrokingColorSpace

(PECL haru >= 0.0.1)

HaruPage::getStrokingColorSpace — 現在の描画の色空間を取得する

説明

int HaruPage::getStrokingColorSpace ( void )

現在の描画の色空間を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

現在の描画の色空間を返します。 返り値の一覧は HaruPage::getFillingColorSpace() と同じです。

エラー / 例外

エラー時に HaruException をスローします。
