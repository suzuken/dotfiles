SQLite3Stmt::paramCount

(PHP 5 >= 5.3.0)

SQLite3Stmt::paramCount — プリペアドステートメント内のパラメータの数を返す

説明

public int SQLite3Stmt::paramCount ( void )

プリペアドステートメント内のパラメータの数を返します。

パラメータ

この関数にはパラメータはありません。

返り値

プリペアドステートメント内のパラメータの数を返します。
