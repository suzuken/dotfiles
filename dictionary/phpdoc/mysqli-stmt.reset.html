mysqli_stmt::reset

mysqli_stmt_reset

(PHP 5)

mysqli_stmt::reset -- mysqli_stmt_reset — プリペアドステートメントをリセットする

説明

オブジェクト指向型

bool mysqli_stmt::reset ( void )

手続き型

bool mysqli_stmt_reset ( mysqli_stmt $stmt )

クライアントおよびサーバで、実行後のプリペアドステートメントをリセットします。

サーバ上のステートメント、mysqli_stmt_send_long_data() で送信したデータ、バッファリングされていない結果、そして現在のエラーをリセットします。 バインド内容や保存された結果セットはクリアされません。保存された結果がクリアされるのは、 プリペアドステートメントを実行した (あるいは閉じた) ときです。

別のクエリを元にプリペアドステートメントを用意する場合は、 mysqli_stmt_prepare() 関数を使用します。

パラメータ

stmt

手続き型のみ: mysqli_stmt_init() が返すステートメント ID。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
