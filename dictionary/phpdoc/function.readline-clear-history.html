readline_clear_history

(PHP 4, PHP 5)

readline_clear_history — ヒストリをクリアする

説明

bool readline_clear_history ( void )

この関数はコマンドラインヒストリ全体をクリアします。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
