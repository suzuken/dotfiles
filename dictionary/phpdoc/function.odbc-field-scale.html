odbc_field_scale

(PHP 4, PHP 5)

odbc_field_scale — フィールドの精度を得る

説明

int odbc_field_scale ( resource $result_id , int $field_number )

指定した ODBC 結果 ID の番号で指定したフィールドの精度を取得します。

パラメータ

result_id

結果 ID。

field_number

フィールド番号。 フィールド番号は、1 から始まります。

返り値

フィールド番号、あるいはエラー時に FALSE を返します。
