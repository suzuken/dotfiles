swf_getfontinfo

(PHP 4)

swf_getfontinfo — フォントの情報を得る

説明

array swf_getfontinfo ( void )

大文字の A と小文字の x の高さをピクセル単位で調べることによって、 フォントについての情報を取得します。

返り値

以下のパラメータを持つ連想配列を返します。

  • Aheight - 大文字 A のピクセル単位の高さ。
  • xheight - 小文字 x のピクセル単位の高さ。
