gmp_sign

(PHP 4 >= 4.0.4, PHP 5)

gmp_sign — 数の符号

説明

int gmp_sign ( resource $a )

数の符号を調べます。

パラメータ

a

GMP 数リソース、 あるいは数値に変換可能な数値形式の文字列。

返り値

a が正の場合に 1、 a が負の場合に -1、そして a がゼロの場合に 0 を返します。

例

例1 gmp_sign() の例

<?php
// 正
echo gmp_sign("500") . "\n";

// 負
echo gmp_sign("-500") . "\n";

// ゼロ
echo gmp_sign("0") . "\n";
?>

上の例の出力は以下となります。

1
-1
0
