ImagickPixelIterator::destroy

ImagickPixelIterator::destroy — PixelIterator に関連付けられているリソースの割り当てを解除する

説明

bool ImagickPixelIterator::destroy ( void )
警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

返り値

成功した場合に TRUE を返します。
