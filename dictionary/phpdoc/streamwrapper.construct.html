streamWrapper::__construct

(PHP 4 >= 4.3.2, PHP 5)

streamWrapper::__construct — 新しいストリームラッパーを作成する

説明

streamWrapper::__construct ( void )

ストリームラッパーをオープンしたときに、 streamWrapper::stream_open() の直前にコールされます。

パラメータ

この関数にはパラメータはありません。
