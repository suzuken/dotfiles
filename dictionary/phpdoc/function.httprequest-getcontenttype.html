HttpRequest::getContentType

(PECL pecl_http >= 0.10.0)

HttpRequest::getContentType — content type を取得する

説明

public string HttpRequest::getContentType ( void )

事前に設定された content type を取得します。

返り値

事前に設定された content type を文字列で返します。
