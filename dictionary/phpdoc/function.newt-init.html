newt_init

(PECL newt >= 0.1)

newt_init — newt を初期化する

説明

int newt_init ( void )

newt インターフェースを初期化します。この関数は、他の newt 関数の前に コールする必要があります。

返り値

成功した場合に 1、失敗した場合に 0 を返します。
