Imagick::setImageOpacity

(PECL imagick 2.0.0)

Imagick::setImageOpacity — 画像の不透明度を設定する

説明

bool Imagick::setImageOpacity ( float $opacity )

画像の不透明度を設定します。このメソッドは、ImageMagick バージョン 6.3.1 以降で Imagick をコンパイルした場合に使用可能です。

パラメータ

opacity

透明度。1.0 が完全な不透明で、0.0 が完全な透明です。

返り値

成功した場合に TRUE を返します。

例

例1 Imagick::setImageOpacity() の例

Imagick::setImageOpacity() を使用する例です。

<?php
/* オブジェクトを作成します */
$image = new Imagick('source.png');

/* 透明度を設定します */
$image->setImageOpacity(0.7);

/* 画像を出力します */
header('Content-type: image/png');
echo $image;

?>
