DOMDocument::createAttribute

(PHP 5)

DOMDocument::createAttribute — 新しい属性を作成する

説明

DOMAttr DOMDocument::createAttribute ( string $name )

この関数は、DOMAttr クラスの新しいインスタンスを作成します。 このノードは、( DOMNode->appendChild() などで) 挿入されない限り、ドキュメント内にあらわれません。

パラメータ

name

属性の名前。

返り値

新しい DOMAttr、あるいはエラーが発生した場合は FALSE を返します。

エラー / 例外

DOM_INVALID_CHARACTER_ERR

name が無効な文字を含んでいる場合に発生します。
