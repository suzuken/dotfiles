ImagickDraw::getFillOpacity

(PECL imagick 2.0.0)

ImagickDraw::getFillOpacity — 描画時の透過度を返す

説明

float ImagickDraw::getFillOpacity ( void )
警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

色やテクスチャによる塗りつぶしの際の透過度を返します。 完全に不透過の場合は 1.0 となります。

返り値

透過度を返します。
