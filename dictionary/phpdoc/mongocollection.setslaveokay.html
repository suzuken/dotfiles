MongoCollection::setSlaveOkay

(PECL mongo >=1.1.0)

MongoCollection::setSlaveOkay — このコレクションの slaveOkay の設定を変更する

説明

public bool MongoCollection::setSlaveOkay ([ bool $ok ] )

読み込みのスレーブへの分散についての情報は、このマニュアルの 問い合わせの節 を参照ください。

パラメータ

ok

可能な限りの問い合わせをレプリカセットのセカンダリメンバに送信したいときに、この Mongo インスタンスを使います。

返り値

このインスタンスの、変更前の slaveOkay の値を返します。
