ReflectionClass::getExtension

(PHP 5)

ReflectionClass::getExtension — 拡張モジュールの情報を取得する

説明

public ReflectionExtension ReflectionClass::getExtension ( void )

拡張モジュールを表す ReflectionExtension オブジェクトを取得します。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

ReflectionExtension オブジェクトを返します。
