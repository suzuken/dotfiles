TokyoTyrant::getIterator — イテレータを取得する

説明

public TokyoTyrantIterator TokyoTyrant::getIterator ( void )

イテレータを取得し、データベース内のすべての キー/値 を反復処理できるようにします。

パラメータ

この関数にはパラメータはありません。

返り値

TokyoTyrantIterator オブジェクトを返します。失敗した場合に TokyoTyrantException をスローします。

例

例1 TokyoTyrant::getIterator() の例

<?php
$tt = new TokyoTyrant("localhost");
$it = $tt->getIterator();

foreach ($it as $k => $v) {

}
?>
