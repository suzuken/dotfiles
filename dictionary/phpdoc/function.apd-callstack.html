apd_callstack

(PECL apd 0.2-0.4)

apd_callstack — 現在のコールスタックを配列で返す

説明

array apd_callstack ( void )

現在のコールスタックを配列形式で返します。

返り値

現在のコールスタックを含む配列を返します。

例

例1 apd_callstack() の例

<?php
print_r(apd_callstack());
?>
