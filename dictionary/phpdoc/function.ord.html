ord

(PHP 4, PHP 5)

ord — 文字の ASCII 値を返す

説明

int ord ( string $string )

string の先頭文字の ASCII 値を返します。

この関数は chr() と逆の動作をします。

パラメータ

string

文字。

返り値

ASCII 値を返します。

例

例1 ord() の例

<?php
$str = "\n";
if (ord($str) == 10) {
    echo "\$str の先頭は改行文字です。\n";
}
?>
