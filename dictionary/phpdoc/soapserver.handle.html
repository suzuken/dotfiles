SoapServer::handle

(PHP 5 >= 5.0.1)

SoapServer::handle — SOAP リクエストを処理する

説明

public void SoapServer::handle ([ string $soap_request ] )

SOAP リクエストを処理し、必要な関数をコールし、レスポンスを返送します。

パラメータ

soap_request

SOAP リクエスト。もし引数が省略された場合、 HTTP リクエストで POST されたデータの中にリクエストがあるとみなします。

返り値

値を返しません。

例

例1 SoapServer::handle() の例

<?php
function test($x)
{
    return $x;
}

$server = new SoapServer(null, array('uri' => "http://test-uri/"));
$server->addFunction("test");
$server->handle();
?>
