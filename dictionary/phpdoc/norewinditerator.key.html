NoRewindIterator::key

(PHP 5 >= 5.1.0)

NoRewindIterator::key — 現在のキーを取得する

説明

public mixed NoRewindIterator::key ( void )

現在のキーを取得します。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

現在のキーを返します。
