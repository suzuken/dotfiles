MongoCursor::explain

(PECL mongo >=0.9.2)

MongoCursor::explain — 最適化やデバッグの際に有効な、クエリの explain を返す

説明

public array MongoCursor::explain ( void )

パラメータ

この関数にはパラメータはありません。

返り値

クエリの explain を返します。

例

例1 MongoCursor::explain() の例

<?php

$cursor = $collection->find(array("x"=>1), array("y"));
$cursor->sort(array("z" => 1))->limit(4)->skip(5);

var_dump($cursor->explain());

?>
