GearmanTask::__construct — Create a GearmanTask instance

説明

GearmanTask::__construct ( void )

Creates a GearmanTask instance representing a task to be submitted to a job server.

パラメータ

この関数にはパラメータはありません。

返り値

A GearmanTask object.
