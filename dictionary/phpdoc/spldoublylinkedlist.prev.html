SplDoublyLinkedList::prev

(PHP 5 >= 5.3.0)

SplDoublyLinkedList::prev — 前のエントリに移動する

説明

void SplDoublyLinkedList::prev ( void )

イテレータを前のノードに移動します。

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。
