SplDoublyLinkedList::key

(PHP 5 >= 5.3.0)

SplDoublyLinkedList::key — 現在のノードのインデックスを返す

説明

mixed SplDoublyLinkedList::key ( void )

この館数は、現在のノードのインデックスを返します。

パラメータ

この関数にはパラメータはありません。

返り値

現在のノードのインデックスを返します。
