MongoDB::forceError()

(バージョン情報なし。おそらく SVN 版にしか存在しないでしょう)

— データベースのエラーを作成する

説明

public bool MongoDB::forceError ( void )

MongoDB を普通に使っている限り、このメソッドはあまり有用ではありません。 このメソッドは、データベースにエラーを起こさせます。つまり、このコマンドを実行した後は、 MongoDB::lastError() が一般的なデータベースエラーを返すということです。

このコマンドは、次のコードと同じ意味です。

<?php

public function forceError() {
    return $this->command(array('forceerror' => 1));
}

?>

パラメータ

この関数にはパラメータはありません。

返り値

データベースからの応答を返します。
