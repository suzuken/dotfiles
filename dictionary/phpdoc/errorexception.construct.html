ErrorException::__construct

(PHP 5 >= 5.1.0)

ErrorException::__construct — 例外を作成する

説明

public ErrorException::__construct() ([ string $message [, int $code [, int $severity [, string $filename [, int $lineno ]]]]] )

例外を作成します。

パラメータ

message

スローする例外メッセージ。

code

例外コード。

severity

例外の深刻度。

filename

例外がスローされたファイル名。

lineno

例外がスローされた行番号。
