TokyoTyrantTable::out

(PECL tokyo_tyrant >= 0.1.0)

TokyoTyrantTable::out — レコードを削除する

説明

public void TokyoTyrantTable::out ( mixed $keys )

テーブルデータベースからレコードを削除します。

パラメータ

keys

整数のキー、あるいはその配列。

返り値

このメソッドは現在のオブジェクトを返します。失敗した場合に TokyoTyrantException をスローします。

例

例1 TokyoTyrantTable::out() の例

<?php
/* テーブルデータベースに接続します */
$tt = new TokyoTyrantTable("localhost", 1979);

/* null を渡して新しい uid を生成します */
$index = $tt->put(null, array("column1" => "some data", "column2" => "more data"));

/* 行を削除します */
$tt->out($index);
?>
