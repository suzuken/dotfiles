mysqli_param_count

(PHP 5)

mysqli_param_count — mysqli_stmt_param_count() のエイリアス

説明

この関数は mysqli_stmt_param_count() のエイリアスです。

注意

注意:

mysqli_param_count() は非推奨で、廃止予定です。

  • mysqli_stmt_param_count() - 指定したステートメントのパラメータ数を返す
