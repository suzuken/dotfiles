fam_open

(PHP 5 <= 5.0.5)

fam_open — FAM デーモンへの接続をオープンする

説明

resource fam_open ([ string $appname ] )

FAM サービスデーモンへの接続をオープンします。

パラメータ

appname

ログ記録用にアプリケーションを特定する文字列。

返り値

成功した場合に FAM サービスへの接続を表すリソース、 失敗した場合に FALSE を返します。

参考

  • fam_close() - FAM 接続を閉じる
