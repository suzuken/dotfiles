yp_err_string

(PHP 4 >= 4.0.6, PHP 5 <= 5.0.5)

yp_err_string — 指定したエラーコードに対応するエラー文字列を返す

説明

string yp_err_string ( int $errorcode )

指定されたエラーコードに対応する エラーメッセージを返します。何が悪かったのかを正確に調べる際に便利です。

パラメータ

errorcode

エラーコード。

返り値

エラーメッセージを表す文字列を返します。

例

例1 NIS エラーの例

<?php
echo "Error: " . yp_err_string(yp_errno());
?>
