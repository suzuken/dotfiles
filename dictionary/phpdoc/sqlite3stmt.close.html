SQLite3Stmt::close

(PHP 5 >= 5.3.0)

SQLite3Stmt::close — プリペアドステートメントを閉じる

説明

public bool SQLite3Stmt::close ( void )

プリペアドステートメントを閉じます。

パラメータ

この関数にはパラメータはありません。

返り値

TRUE を返します。
