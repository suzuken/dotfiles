PDF_setmiterlimit

(PHP 4, PECL pdflib >= 1.0.0)

PDF_setmiterlimit — miter limit を設定する

説明

bool PDF_setmiterlimit ( resource $pdfdoc , float $miter )

miter limit を設定します。成功した場合に TRUE を、失敗した場合に FALSE を返します。
