ReflectionFunctionAbstract::isUserDefined

(PHP 5)

ReflectionFunctionAbstract::isUserDefined — ユーザ定義関数であるかどうかを調べる

説明

public bool ReflectionFunctionAbstract::isUserDefined ( void )

この関数がユーザ定義関数なのか内部関数なのかを調べます。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

ユーザ定義関数である場合に TRUE、それ以外の場合に FALSE を返します。
