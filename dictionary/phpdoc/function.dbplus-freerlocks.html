dbplus_freerlocks

(PHP 4 <= 4.1.0, PECL dbplus >= 0.9)

dbplus_freerlocks — 指定したリレーションに関する全てのタプルロックを解放する

説明

int dbplus_freerlocks ( resource $relation )

指定した relation で保持されている全てのタプルロックを解放します。

パラメータ

relation

dbplus_open() でオープンしたリレーション。

返り値

注意
