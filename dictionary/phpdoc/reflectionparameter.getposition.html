ReflectionParameter::getPosition

(PHP 5 >= 5.2.3)

ReflectionParameter::getPosition — パラメータの位置を取得する

説明

public int ReflectionParameter::getPosition ( void )

パラメータの位置を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

パラメータの位置を取得します。左から右へ数え、最初のパラメータが 0 番目となります。
