ReflectionFunctionAbstract::getStartLine

(PHP 5)

ReflectionFunctionAbstract::getStartLine — 開始行番号を取得する

説明

public int ReflectionFunctionAbstract::getStartLine ( void )

関数の開始行番号を取得します。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

開始行番号を返します。
