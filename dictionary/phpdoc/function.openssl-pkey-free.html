openssl_pkey_free

(PHP 4 >= 4.2.0, PHP 5)

openssl_pkey_free — 秘密鍵を開放する

説明

void openssl_pkey_free ( resource $key )

この関数は、openssl_pkey_new() で作成した秘密鍵を開放します。

パラメータ

key

キーを保持するリソース。

返り値

値を返しません。
