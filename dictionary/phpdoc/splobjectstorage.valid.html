SplObjectStorage::valid

(PHP 5 >= 5.1.0)

SplObjectStorage::valid — イテレータの現在のエントリが有効かどうかを返す

説明

public bool SplObjectStorage::valid ( void )

イテレータの現在のエントリが有効かどうかを返します。

パラメータ

この関数にはパラメータはありません。

返り値

イテレータのエントリが有効な場合に TRUE、それ以外の場合に FALSE を返します。

例

例1 SplObjectStorage::valid() の例

<?php
$s = new SplObjectStorage();

$o1 = new StdClass;
$o2 = new StdClass;

$s->attach($o1, "d1");
$s->attach($o2, "d2");

$s->rewind();
while($s->valid()) {
    echo $s->key()."\n";
    $s->next();
}
?>

上の例の出力は、 たとえば以下のようになります。

0
1
