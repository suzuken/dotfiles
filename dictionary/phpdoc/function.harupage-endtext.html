HaruPage::endText

(PECL haru >= 0.0.1)

HaruPage::endText — 現在のテキストオブジェクトを終了する

説明

bool HaruPage::endText ( void )

現在のテキストオブジェクトを終了します。

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合に TRUE を返します。

エラー / 例外

エラー時に HaruException をスローします。
