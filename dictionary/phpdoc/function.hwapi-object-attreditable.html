hw_api_object->attreditable

(PHP 4, PHP 5 < 5.2.0, PECL hwapi SVN)

hw_api_object->attreditable — 属性が編集可能かどうかを調べる

説明

bool hw_api_object::attreditable ( array $parameter )

属性が編集可能かどうかを調べます。

パラメータ

parameter

返り値

属性が編集可能な場合に TRUE、それ以外の場合に FALSE を返します。
