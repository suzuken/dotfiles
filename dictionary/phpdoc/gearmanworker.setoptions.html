GearmanWorker::setOptions

(PECL gearman >= 0.5.0)

GearmanWorker::setOptions — Set worker options

説明

public bool GearmanWorker::setOptions ( int $option )

Sets one or more options to the supplied value.

パラメータ

option

The options to be set

返り値

Always returns TRUE.
