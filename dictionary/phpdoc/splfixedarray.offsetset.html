SplFixedArray::offsetSet

(PHP 5 >= 5.3.0)

SplFixedArray::offsetSet — 指定した添字の新しい値を設定する

説明

public void SplFixedArray::offsetSet ( int $index , mixed $newval )

index で指定した添字の値を newval に設定します。

パラメータ

index

設定したい添字。

newval

index の新しい値。

返り値

値を返しません。
