cosh

(PHP 4 >= 4.1.0, PHP 5)

cosh — 双曲線余弦（ハイパボリックコサイン）

説明

float cosh ( float $arg )

arg のハイパボリックコサインを返します。 これは、(exp(arg) + exp(-arg))/2 で定義されます。

パラメータ

arg

処理する引数。

返り値

arg のハイパボリックコサインを返します。

参考

  • sinh() - 双曲線正弦（ハイパボリックサイン）
