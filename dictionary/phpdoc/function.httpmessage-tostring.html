HttpMessage::toString

(PECL pecl_http >= 0.10.0)

HttpMessage::toString — 文字列表現を取得する

説明

public string HttpMessage::toString ([ bool $include_parent = false ] )

メッセージの文字列表現を取得します。

パラメータ

include_parent

返される文字列に親メッセージを含めるかどうかを指定します。

返り値

メッセージを文字列として返します。
