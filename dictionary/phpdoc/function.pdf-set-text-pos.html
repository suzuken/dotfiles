PDF_set_text_pos

(PHP 4, PECL pdflib >= 1.0.0)

PDF_set_text_pos — テキストの位置を設定する

説明

bool PDF_set_text_pos ( resource $p , float $x , float $y )

テキストの出力位置を設定します。成功した場合に TRUE を、失敗した場合に FALSE を返します。
