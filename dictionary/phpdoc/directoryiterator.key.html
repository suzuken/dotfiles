DirectoryIterator::key

(PHP 5)

DirectoryIterator::key — 現在の DirectoryIterator アイテムのキーを返す

説明

public string DirectoryIterator::key ( void )

現在の DirectoryIterator アイテムのキーを取得します。

パラメータ

この関数にはパラメータはありません。

返り値

現在の DirectoryIterator アイテムのキーを返します。

例

例1 DirectoryIterator::key() の例

<?php
$dir = new DirectoryIterator(dirname(__FILE__));
foreach ($dir as $fileinfo) {
    if (!$fileinfo->isDot()) {
        echo $fileinfo->key() . " => " . $fileinfo->getFilename() . "\n";
    }
}
?>

上の例の出力は、 たとえば以下のようになります。

0 => apple.jpg
1 => banana.jpg
2 => index.php
3 => pear.jpg
