mb_ereg_match

(PHP 4 >= 4.2.0, PHP 5)

mb_ereg_match — マルチバイト文字列が正規表現に一致するか調べる

説明

bool mb_ereg_match ( string $pattern , string $string [, string $option = "msr" ] )

マルチバイト文字列に対する正規表現マッチングを行います。

パラメータ

pattern

正規表現パターン。

string

評価する文字列。

option

返り値

string が正規表現 pattern に一致する場合にTRUE 、 そうでない場合に FALSE を返します。

注意

注意:

内部エンコーディングあるいは mb_regex_encoding() で指定した文字エンコーディングを、 この関数の文字エンコーディングとして使用します。

参考

  • mb_regex_encoding() - 現在の正規表現用のエンコーディングを文字列として返す
  • mb_ereg() - マルチバイト文字列に正規表現マッチを行う
