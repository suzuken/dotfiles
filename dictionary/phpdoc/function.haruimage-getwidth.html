HaruImage::getWidth

(PECL haru >= 0.0.1)

HaruImage::getWidth — 画像の幅を取得する

説明

int HaruImage::getWidth ( void )

画像の幅を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

画像の幅を返します。

エラー / 例外

エラー時に HaruException をスローします。
