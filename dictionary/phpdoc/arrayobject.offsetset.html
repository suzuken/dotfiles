ArrayObject::offsetSet

(PHP 5 >= 5.0.0)

ArrayObject::offsetSet — 指定したインデックスに新しい値をセットする

説明

void ArrayObject::offsetSet ( mixed $index , mixed $newval )

指定したインデックスに新しい値を設定します。

パラメータ

index

設定したいインデックス。

newval

index の新しい値。

返り値

値を返しません。

例

例1 ArrayObject::offsetSet() の例

<?php
class Example {
    public $property = 'prop:public';
}
$arrayobj = new ArrayObject(new Example());
$arrayobj->offsetSet(4, 'four');
$arrayobj->offsetSet('group', array('g1', 'g2'));
var_dump($arrayobj);

$arrayobj = new ArrayObject(array('zero','one'));
$arrayobj->offsetSet(null, 'last');
var_dump($arrayobj);
?>

上の例の出力は以下となります。

object(ArrayObject)#1 (3) {
  ["property"]=>
  string(11) "prop:public"
  [4]=>
  string(4) "four"
  ["group"]=>
  array(2) {
    [0]=>
    string(2) "g1"
    [1]=>
    string(2) "g2"
  }
}
object(ArrayObject)#3 (3) {
  [0]=>
  string(4) "zero"
  [1]=>
  string(3) "one"
  [2]=>
  string(4) "last"
}
