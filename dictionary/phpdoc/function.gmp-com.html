gmp_com

(PHP 4 >= 4.0.4, PHP 5)

gmp_com — 1 の補数を計算する

説明

resource gmp_com ( resource $a )

a について、1 の補数を返します。

パラメータ

a

GMP 数リソース、 あるいは数値に変換可能な数値形式の文字列。

返り値

a についての 1 の補数を GMP 数で返します。

例

例1 gmp_com() の例

<?php
$com = gmp_com("1234");
echo gmp_strval($com) . "\n";
?>

上の例の出力は以下となります。

-1235
