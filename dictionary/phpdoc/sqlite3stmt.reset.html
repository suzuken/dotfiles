SQLite3Stmt::reset

(PHP 5 >= 5.3.0)

SQLite3Stmt::reset — プリペアドステートメントをリセットする

説明

public bool SQLite3Stmt::reset ( void )

プリペアドステートメントをリセットし、実行前の状態に戻します。 バインドした内容は、リセットしてもそのまま残ります。

パラメータ

この関数にはパラメータはありません。

返り値

ステートメントのリセットに成功した場合に TRUE、 失敗した場合に FALSE を返します。
